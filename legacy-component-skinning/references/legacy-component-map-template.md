## 使用方式
1. 先按页面或模块盘点所有核心组件。
2. 明确每个 legacy 组件的样式入口和共享层可改点。
3. 标记哪些能在共享层解决，哪些只能页面局部补丁。
4. 实施时优先处理覆盖面最大的共享组件。
